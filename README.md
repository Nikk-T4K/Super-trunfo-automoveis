# Super Trunfo - Automóveis

Jogo de cartas interativo baseado no clássico "Super Trunfo", desenvolvido em Python para consolidação de conceitos de lógica algorítmica, controle de fluxos e manipulação dinâmica de estruturas de dados.

## Modos de Jogo
- **Single Player:** O usuário enfrenta o computador, que possui uma inteligência algorítmica básica para tomada de decisões automatizada.
- **Multiplayer:** Modo local para disputa entre dois jogadores em turnos alternados.

## Detalhes Técnicos
- Estruturação do baralho utilizando listas e matrizes em Python.
- Embaralhamento e distribuição automatizada com a biblioteca `random`.
- Mecânica de turnos dinâmicos com controle de "monte de espera" (para cartas acumuladas em caso de empate).
